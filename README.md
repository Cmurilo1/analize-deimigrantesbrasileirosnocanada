
# 📊 Análise de Imigração: Brasil -> Canadá

Este projeto é um dashboard interativo desenvolvido em Python e Flask que visualiza os dados históricos de imigração do Brasil para o Canadá utilizando as bibliotecas Pandas e Plotly.

## 🛠️ Tecnologias Utilizadas
- Python 3
- Flask (Framework Web)
- Pandas (Tratamento de dados)
- Plotly (Gráficos interativos)
- Bootstrap 5 (Design responsivo)

## 🚀 Como Rodar o Projeto Localmente
1. Clone o repositório:
   git clone https://github.com/Cmurilo1/analize-deimigrantesbrasileirosnocanada.git
2. Instale as dependências:
   pip install -r requirements.txt
3. Execute a aplicação:
   python api/index.py
4. Acesse no navegador: http://127.0.0.1:5000

## 📈 Visualização
O projeto gera um gráfico de linhas interativo que permite acompanhar a evolução anual do fluxo migratório, facilitando a identificação de picos e tendências.
